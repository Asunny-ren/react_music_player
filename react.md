# react

``` dash
# 利用脚手架搭建app
npx create-react-app music-app

# 安装stylus 和stylus loader
npm install stylus stylus-loader --save-dev

# 喷射webpack配置,在webpack.config.dev.js中添加stylus-loader配置
npm run eject

# 启动
npm start

# 打包
npm run build
```
